# base64-image-converter

Base64-Image-Converter is a Windows UI written in golang to convert images into base64 text.

Screenshot:

![screenshot](./img/screenshot.png "Screenshot")



## Installation


```bash
go run .
```

## Usage

```bash
go build .
```
